# Mausamkahaal

Fetches the weather forecast for next 7 days from george-vustrey-weather and mashape api.

## Installation

Add this line to your application's Gemfile:

    gem 'mausamkahaal'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mausamkahaal
